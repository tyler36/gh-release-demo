## Required

- PHP >= 8.0

## Usage

- Clone repo

    ```shell
    git clone https://github.com/tyler36/gh-release-demo
    ```

- Run following to commit first release

    ```shell
    php vendor/bin/conventional-changelog --first-release
    ```

- Commit directly to main
